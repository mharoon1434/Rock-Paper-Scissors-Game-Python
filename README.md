<h1>Rock Paper Scissors (Python)</h1>

<p>A simple command-line Rock Paper Scissors game built using Python.<br>
The user plays against the computer, which makes a random choice.</p>

<h2>Features</h2>
<ul>
  <li>User input for rock, paper, or scissors</li>
  <li>Random computer choice</li>
  <li>Handles win, lose, and tie conditions</li>
  <li>Beginner-friendly Python project</li>
</ul>

<h2>How It Works</h2>
<ol>
  <li>The user enters their choice (rock, paper, or scissors)</li>
  <li>The computer randomly selects one option</li>
  <li>The program compares both choices</li>
  <li>The result (win, lose, or tie) is displayed</li>
</ol>


<h2>How to Run</h2>
<ol>
  <li>Clone this repository:
    <pre><code>git clone https://github.com/mharoon1434/Rock-Paper-Scissors-Game-Python.git</code></pre>
  </li>
  <li>Navigate to the project folder:
    <pre><code>cd Rock-Paper-Scissors-Game-Python</code></pre>
  </li>
  <li>Run the script:
    <pre><code>python game.py</code></pre>
  </li>
</ol>
